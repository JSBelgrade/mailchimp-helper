# mailchimp-helper
Handles subscription to MailChimp because their JS API sucks

## How to install

### Prerequests

1. io.js or node.js with `--harmony` flag
2. `MAILCHIMP_API_KEY`, `NODESCHOOL_LIST_ID` and `NODESCHOOL_MENTOR_LIST_ID` in 
process env variables.

### Steps

1. Run `npm install`
2. Run `node index.js`
